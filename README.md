# Contact Form System - ASP.NET Core MVC

## Project Overview

This project is a Contact Form System developed using ASP.NET Core MVC.

The application allows users to:

* Enter Name
* Enter Email
* Enter Message
* Submit the form
* Validate user input
* Display submitted data
* Show success message after submission

---

## Concepts Covered

### Forms

Used HTML Forms to collect user input.

### GET Request

Displays the Contact Form page.

### POST Request

Handles form submission and processes user data.

### Model Binding

Automatically maps form values to the ContactModel object.

### Validation

Implemented Data Annotations for required fields and email validation.

### ViewBag

Used ViewBag to display submitted information and success messages.

### Bootstrap UI

Used Bootstrap components for responsive and professional design.
